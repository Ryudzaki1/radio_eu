# Scripts

This folder keeps only current project scripts at the top level.

- `smoke-tests.js` - fast local/container smoke checks.
- `ops/` - reusable operational scripts for checking or deploying the current RU/EU setup.
- `legacy/` - one-off setup, diagnostics, and historical repair scripts kept for reference.

Do not run `legacy/` scripts against production unless you have reviewed the file and the exact target first.
