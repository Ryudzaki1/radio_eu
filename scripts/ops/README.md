# Operational scripts

Reusable helpers for the current two-server architecture:

- `check_eu_bot_runtime.py` - inspect the EU Telegram bot runtime.
- `check_radio_servers_state.py` - summarize RU/EU server state.
- `check_ru_app_health.py` - check the RU radio app health.
- `deploy_eu_telegram_bot.py` - deploy the EU bot role.
- `set_ru_public_url_ip.py` - update RU public URL/IP related runtime state.
- `upload_music_to_ru.py` - upload music files to the RU server.
